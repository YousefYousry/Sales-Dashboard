# Sales Dashboard - Excel Project

This project is my final Excel project as part of the Digital Egypt Pioneers Initiative (DEPI) training.

The dashboard analyzes sales data and transforms raw datasets into interactive insights that help understand business performance.

## Project Overview
This Sales Dashboard provides a clear view of:
- Total Sales Performance
- Sales by Year
- Product Line Analysis
- Deal Size Distribution
- Order Status Insights

## Key Insights
- Total Sales: 4.68M
- Highest Sales Year: 2004
- Top Product Line: Classic Cars
- 92% of orders were successfully shipped
- Most transactions were Medium and Small deal sizes
